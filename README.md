### 📝 وصف البرنامج
برنامج **D3M0N3T** هو أداة متطورة مخصصة لـRoblox Exploters، يوفر بيئة عمل سريعة ومستقرة لتنفيذ وتحرير السكربتات البرمجية باستخدام محرك **XenoUI** واجهة مستخدم عصرية تعتمد على تقنية **WebView2**.

### 🚀 المميزات الجديدة في الإصدار 5.4
* **هوية بصرية جديدة**: اعتماد شعار **D5** البنفسجي المحدث بتصميم هندسي مائل.
* **تبويب About**: إضافة صفحة تعريفية متصلة بالإنترنت لعرض أخبار الفريق وحقوق النشر.
* **إصلاح الأيقونات**: تم حل مشكلة "Cache" الأيقونة في شريط المهام بنجاح.
* **تحسين الأداء**: استجابة أسرع للمحرر عند التعامل مع السكربتات الطويلة.

---

### 🛠️ متطلبات التشغيل
لضمان عمل البرنامج بكفاءة، يرجى التأكد من توفر:
1. **نظام التشغيل**: Windows 10 / 11.
2. **WebView2 Runtime**: (ضروري لتشغيل Hub و Keeper).
3. **.NET Framework**: الإصدار 4.7.2 أو أحدث.

---

### 📂 محتويات المجلد
* `D3M0N3T.exe`: الملف التنفيذي الأساسي للبرنامج.
* `D5.ico`: ملف الأيقونة البنفسجية الخاص بالبرنامج.
* `autorun.inf`: ملف تعريف الأيقونة للمجلد.
