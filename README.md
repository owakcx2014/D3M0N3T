# 💜 D3M0N3T Framework v6.4 🚀

![D3M0N3T Logo](D5.png)

**D3M0N3T** هو إطار عمل (Framework) متطور وقوي لبيئة **Roblox**، تم بناؤه لتقديم أقصى درجات الأداء مع واجهة مستخدم (UI) فريدة وعصرية تركز على السرعة والبساطة.

---

### 🌟 المميزات الجديدة في الإصدار v6.4
* 🎞️ **Splash Screen متطورة:** شاشة ترحيبية سينمائية تظهر عند تشغيل البرنامج لتعزيز تجربة المستخدم.
* 📁 **دعم ملفات TXT:** إمكانية رفع وتشغيل السكربتات بصيغة `.txt` بجانب `.lua` و `.luau`.
* 🎨 **واجهة مستخدم محسنة:** إعادة تنظيم تاب "About" ليكون أكثر وضوحاً وتنظيماً بهوية D3M0N3T الخاصة.
* 🌐 **نظام الروابط الخارجية:** الآن تفتح جميع الروابط (Discord/GitHub) في متصفحك الافتراضي مباشرة لضمان الأمان.
* 🚀 **تحسين الحاقن (Injector):** زيادة سرعة الاستجابة عند الضغط على زر الربط (Attach).

---

### 🛠️ طريقة الاستخدام
1. افتح **D3M0N3T.exe** (ستظهر لك شاشة الترحيب أولاً).
2. استخدم زر **Attach** (🔗) للربط باللعبة.
3. ارفع السكربت الخاص بك عبر **Upload** (📤) أو الصقه يدوياً.
4. اضغط **Execute** (🚀) وانطلق!
